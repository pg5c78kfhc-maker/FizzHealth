# Decision Intelligence Platform Sprint

## FH-1058 — Decision Intelligence Engine
Implemented central evaluators for nutrient priority, Chef recommendations, LDL Support, Steps, maintenance calories, and restaurant recommendations. `evaluateDecision(type, payload)` provides one public dispatch API.

## FH-1059 — Decision Trace Framework
All central evaluators return the same versioned DecisionTrace contract: decision type, subject ID/name, score, confidence, status, rank, factors, positive/negative contributors, missing inputs, methodology, data used, projected result, action, timestamp, engine version, and rules version.

## FH-1060 — Decision Simulator
Implemented an in-memory What If simulator for hypothetical meal nutrition and additional steps. It reruns nutrient, LDL, and Steps decisions, shows before/after effects, and explicitly records `actualDataChanged: false`.
