# FH-1061 — Decision Trace UI

## Definition of Done

- [x] Every canonical DecisionTrace can be rendered in a full-screen disclosure view.
- [x] Score, confidence, status, rank, subject identity, engine version, rules version, and evaluation time are visible.
- [x] Every recorded factor is shown and grouped by decision category.
- [x] Positive, negative, and net factor impacts are explicit.
- [x] Evidence used, evaluated inputs, projected effects, comparisons, missing information, and recommended action are visible.
- [x] Raw immutable trace is available for audit without changing the decision.
- [x] Malformed traces fail safely with a visible validation error.
- [x] Presentation helpers have automated tests.
- [x] Full test suite passes.
- [x] Production build passes.
