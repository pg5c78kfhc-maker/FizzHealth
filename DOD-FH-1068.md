# Definition of Done — FH-1068

- Generates selectable 15-, 20-, and 30-day forward plans from verified foods currently on hand.
- Never schedules more servings than verified inventory supports.
- Excludes expired and unavailable foods.
- Rotates foods to avoid unnecessary repetition while still prioritizing expiration and nutrition fit.
- Shows days covered, inventory coverage, verified servings, and projected runout date.
- Allows the generated plan to be added to the existing planned-meals calendar.
- Automated tests cover inventory limits, rotation, and expired-food exclusion.
- Full test, release verification, and production build pass.
