# ADR-006 — Context-aware action routing

Date: 2026-07-20
Status: Accepted
Related: FH-1260

## Decision

Decision explanation pages remain explanatory. Their primary action routes to the workflow that can resolve the condition rather than always opening Universal Capture.

- Calories remaining → meal planner / Chef’s Recommendations.
- Steps action → Steps metric entry.
- Pantry attention → exact flagged pantry items.
- Restaurant review → Restaurant Intelligence.

## Preserved behavior

The LDL Support, Steps, and Estimated Maintenance explanation layouts are not modified. This release changes destinations, not the explanation framework.
